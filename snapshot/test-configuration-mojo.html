errorprone:test-configuration

Full name:

io.github.jonasrutishauser.maven:errorprone-maven-plugin:1.0.0-SNAPSHOT:test-configuration

Description:

No description.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: process-test-resources.

Optional Parameters
